原版界面调整小记

前言

早上差点儿起不来，毕竟昨天奋战那么久想用next，今天打开之后发现原版的也挺不错的，要不咬咬牙强迫自己调个界面吧。

首先打算从底部着手，毕竟整体布局上都是常规化，只是这个脚太难看了。

这边先观察下整体，发下其实居中就凑合能看了，就默默的在footer里面找了下

text-align: center;
line-height: 20px;

真是感觉调界面这东西不适合我，个人偏向于后端代码以及js相关的，css的话能调但是不喜欢，从来公司之初到现在后端与js
摸索的不少，css感觉就是个新手，额 应该说是菜鸟-_-||。

调好了底部（凑合着看吧），也对右侧那一个个大方框的背景色淡化了下（虽说不明显），一看滚动条，默认的滚动条说实话看多了感觉凑合，可是到这一看受不鸟，网上搜了下，原来千言万语汇成一段话。

/* 设置滚动条的样式 */
::-webkit-scrollbar {
    width: 12px;
}
/* 滚动槽 */
::-webkit-scrollbar-track {
    -webkit-box-shadow: inset 0 0 6px #5FB878;
    /*border-radius: 10px;*/
}
/* 滚动条滑块 */
::-webkit-scrollbar-thumb {    
    /*border-radius: 10px;*/    
    background: #5FB878;   
-webkit-box-shadow: 
    inset 0 0 6px #5FB878;
}
::-webkit-scrollbar-thumb:window-inactive {    
    background: #5FB878;
}

ps:注释的只是圆角而已，看着效果不好就毙了。

之后又试了下缩进，发现原版的缩进更是受不了，在网上搜了下替换了blockquote的样式。

font: 14px/22px normal helvetica, sans-serif;
margin-top: 10px;
margin-bottom: 10px;
/*margin-left: 50px;*/
padding-left: 15px;
padding-top: 10px;
padding-right: 10px;
padding-bottom: 10px;
border-left: 3px solid #ccc;
background-color:#f1f1f1

折腾了好久发现不能缩进，看别人的代码段都可以为啥我这不行，一搞发现，妈的缩进是>,代码段是三个`，也是服了。
